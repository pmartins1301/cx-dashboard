# CX Dashboard

Dashboard profissional de Customer Experience criado com React, Vite, Tailwind CSS, Recharts e Lucide React.

## Como rodar

```bash
npm install
npm run dev
```

Depois, abra a URL exibida no terminal.

## O que esta incluido

- Header com CX Intelligence e logo da NLT Telecom
- Filtros por periodo: hoje, 7 dias, 30 dias e trimestre
- Cards de KPI para atendimentos, SLA, CSAT, TMA, pendencias e reincidencia
- Graficos de atendimentos por dia, motivos técnicos, canais e SLA por atendente
- Tabela de performance dos atendentes da operação de suporte técnico
- Secao destacada de Insights da IA
- Recomendacoes da IA
- Area Pergunte aos dados com respostas simuladas baseadas nos indicadores
- Avaliacao por canal: WhatsApp, ligacoes, e-mail e chat
